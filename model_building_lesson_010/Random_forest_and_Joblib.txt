🌲 Random Forest (brief)

Random Forest is an ensemble ML algorithm that combines many decision trees to make better predictions.

Each tree is trained on random samples of the data (bagging)
Each split uses a random subset of features
Final output = majority vote (classification) or average (regression)
✅ Why use it
Reduces overfitting compared to a single tree
Works well on tabular data
Minimal preprocessing (no scaling needed)



example>>>>>>>>>>>>>>>>>>>>>>>>

from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X_train, y_train)

preds = model.predict(X_test)



💾 joblib (brief)

joblib is a Python library used to save and load trained models efficiently.

👉 Very useful in ML when you want to:

Reuse trained models
Deploy models (Flask, APIs)
Avoid retraining every time



Save a model
import joblib

joblib.dump(model, "rf_model.pkl")
⚡ Load a model
model = joblib.load("rf_model.pkl")
🔥 Why joblib over pickle?
Faster for large NumPy arrays
Optimized for scikit-learn models
Handles big data better
💡 Typical workflow (real-world)
# train
model.fit(X_train, y_train)

# save
joblib.dump(model, "model.pkl")

# later...
model = joblib.load("model.pkl")
preds = model.predict(new_data)
